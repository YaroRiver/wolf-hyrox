## GREEN version — Option A (default): Extended entry run

### Warm-up
- Bike 8 min easy, HR < 120
- Mobility 5 min
- BBJ check: 3 controlled reps
- 10 wall balls @ 9 kg (shoulder + chalk)

### Main fragment — 4 rounds
Each round:
- 400 m run @ 4:45–5:05 / km (controlled, not lazy)
- Walking Lunges 20–22 m @ 28+28 kg DB
- BBJ 20 m @ controlled rhythm (R1 ≈ 1:20–1:30; apply lesson from Saturday)
- Wall Balls 25 reps @ 9 kg, target 58–68 sec
- Rest: 90 sec (R1–R3); 2:00 (after R4 → transition)

### Final entry run — 1.2 km (harder than W20)
- Pace 4:55–5:10 / km
- HR target before final WB: 145–160 bpm
- If HR > 163 and breathing chaotic: 30 sec walk, controlled restart
- If HR < 140: extend run by 200 m

### Transition discipline
- 60–90 sec walk after 1.2 km
- 5–8 sec pause before first WB rep
- Chalk hands
- Two controlled breaths

### Ceiling test — WB AMRAP clean
- Floor: **50 reps** (= W20 confirmed; consolidation pass)
- Target: **50 clean** (harder conditions = same result is a progression)
- Stretch: **55+** (if calves and breathing remain clean)
- Stop rule: technical failure (throw height, catch, squat depth, torso, panic breathing)
- 1 ugly rep = stop
